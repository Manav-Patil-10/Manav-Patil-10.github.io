Features

Everything your AI assistant should do

Built in 8 progressive phases, each adding a new capability.

🧠

Persistent Memory

Remembers every conversation across sessions using SQLite.

📚

Personal Knowledge Base

Ask questions about your own notes — powered by RAG.

✅

Natural Language Tasks

Manage your to-do list through conversation.

🔀

Multi-Agent Orchestration

Routes requests to the right specialist agent automatically.

🌐

Web Search

Automatically searches the web for current information.

🔒

Privacy Mode

Fully local model with one flag — zero data leaves your machine.

📋

Proactive Briefings

Daily briefing generated automatically every morning.

💻

Web Dashboard

Clean browser interface — no terminal required.

Get started

Up and running in 3 steps

1

Clone the repo

Download the project to your computer.

git clone https://github.com/Manav-Patil-10/personal-ai-os.git
2

Run setup

Installs dependencies and sets up your API key.

python setup.py
3

Start using it

Launch the web dashboard.

python app.py
